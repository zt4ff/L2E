# Data Extraction Template

## Unstructured Input 1

"John Doe, age 29, lives in Paris and works as a software engineer."

## Template Prompt

"Extract the following fields from the text: Name, Age, Location, Occupation. Return the output in JSON format."

## Output

```json
{
  "Name": "John Doe",
  "Age": 29,
  "Location": "Paris",
  "Occupation": "Software Engineer"
}
```

## Unstructured Input 2

"Maria Gomez is 34 years old. She resides in Madrid and is employed as a data analyst."

## Output

```json
{
  "Name": "Maria Gomez",
  "Age": 34,
  "Location": "Madrid",
  "Occupation": "Data Analyst"
}
```

## Unstructured Input 3

## Reflection

The template ensures structured, consistent JSON output across varied sentence structures. Field names remain stable.
