# Step-by-Step Reasoning Prompt

## Coding problem

Create a function combos, that accepts a single positive integer num (30 > num > 0) and returns an array of arrays of positive integers that sum to num.

### AI's response

```js
function combos(num) {
  const result = [];

  function backtrack(start, remaining, path) {
    if (remaining === 0) {
      result.push([...path]);
      return;
    }

    for (let i = start; i <= remaining; i++) {
      path.push(i);
      backtrack(i, remaining - i, path);
      path.pop();
    }
  }

  backtrack(1, num, []);
  return result;
}
```

---

## With reasoning prompt

To build `combos(num)`, we need all combinations of positive integers that sum to `num`.

Key ideas:

1. Each combination must sum to `num`.
2. All numbers must be positive integers.
3. Order should not create duplicates.
   - `[1,2]` and `[2,1]` should count as one combination.

4. We can enforce this by always choosing numbers ≥ the last chosen number.

Step-by-step strategy:

- Use recursion.
- Track:
  - `remaining` sum
  - current `path`
  - `start` number to prevent duplicates

- If `remaining === 0`, push the path.
- Otherwise, try all numbers from `start` to `remaining`.

Final function:

```js
function combos(num) {
  const result = [];

  function backtrack(start, remaining, path) {
    if (remaining === 0) {
      result.push([...path]);
      return;
    }

    for (let i = start; i <= remaining; i++) {
      path.push(i);
      backtrack(i, remaining - i, path);
      path.pop();
    }
  }

  backtrack(1, num, []);
  return result;
}
```

## Comparison and Reflection

With reflection, without reasoning, the AI just aimed to provided the code solution without step-by-step reasoning to how it came with such solutions. The prompt with reasoning instructions did this, provided step-by-step approach to solving a problem. As learners, this is essential to build a reasoning pattern and approach to things.

Also, in past previous experience, I've noticed that asking AI to reason in many cases helps the AI to identify the flaw in it's approach and backtrace.
