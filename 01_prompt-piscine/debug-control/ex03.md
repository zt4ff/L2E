# Using Delimiters to Define Context

## Prompt without delimiter request

```txt
Who is the father of programing? ### What is 2 + 2? ### At what year did programming become mainstream?
```

### Output

Who is the father of programming?

Most people credit Ada Lovelace.
In 1843, she wrote what is considered the first algorithm intended for a machine, the Analytical Engine.

Others sometimes mention Alan Turing, who laid the theoretical foundation of modern computing.

What is 2 + 2?

4

At what year did programming become mainstream?

Programming became mainstream in the 1980s and 1990s, when personal computers like the IBM Personal Computer and the Apple Macintosh entered homes and offices.

It expanded further in the 2000s with the internet and mobile apps.

## Prompt with delimiter note

```txt
Who is the father of programing? ### What is 2 + 2? ### At what year did programming become mainstream? Only answer question bounded by the delmiter "###"
```

### Output

4

## Comparison

The prompt without delimiter request proceeded to answer all the questions, while the one with clear instruction to only answer the question bounded by the delimiter, "###", proceeded to answer the one question bounded.
