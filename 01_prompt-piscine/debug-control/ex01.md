# Reducing False or Invented Facts

## Prompt with Error

### Prompt:

`Give the capital of five imaginary states in Nigeria`

### Output:

Here are five imaginary Nigerian states and their capitals:

- Ajoro State – Capital: Temidire

- Zamora State – Capital: Kantuga

- Obanta State – Capital: Ilumara

- Karimi State – Capital: Danjeko

- Ekanem State – Capital: Ufotara

## Corrected Prompt

### Prompt:

`List five states in Nigeria with their capital`

### Output:

Here are five real states in Nigeria and their capitals:

- Lagos State – Ikeja

- Kano State – Kano

- Rivers State – Port Harcourt

- Oyo State – Ibadan

- Enugu State – Enugu

## Comparison

The error prompt gave out imaginery output as instructed but the 'correct' prompt gave out factual and verified information.
