Explain AI with mathematical foundations.

## Selected AI model

Gemini 3 Flash

## Testing

### Overview of prompt with temperature = 0.2

The response is

- highly conservative
- focused
- precise

### Overview of prompt with temperature = 0.9

The response is

- very creative - the AI presented the formulas and probability for Stylistically
- imaginative
- creative

### Overview of prompt with top-p values = 0.5

- restricts the model to only consider the top-performing tokens that make up 50% of the total probability, forcing more predictable and focused output.

### Overview of prompt with top-p values = 1

- instructs the model to consider the entire pool of tokens (100%), allowing for maximum diversity and creativity.
