# Role-Based Mentoring and Feedback

## Ask the model to act as a coding mentor.

**Prompt:** Act like my coding mentor for this conversation

## Provide a simple piece of code (e.g., a Python function with a bug).

```py
def checkPalindrome(input):
    reversedInput = input[-1:0:0]
    if input == reversedInput:
        return True
    return False
```

## Prompt the model: "Act as a mentor reviewing this code. Provide corrections and explain improvements." - Summary of response

- slicing is wrong
- avoid shadowing built-ins
- simplify the logic
- handle edge cases

## Review

Role-based prompting helped the AI to tailor it's response in a way that is helpful and step-by-step feedback. Not just straight-up correcting the code but by listing the problems, providing feedback and how to correct the bug.
