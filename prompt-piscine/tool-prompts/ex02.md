# Simulated API Call (Weather Example)

## Prompt:

"Given the weather json format:

```json
{
  "location": "string",
  "temperature_celsius": "number",
  "condition": "string",
  "humidity_percent": "number",
  "wind_kph": "number",
  "forecast": [
    {
      "time": "string",
      "temperature_celsius": "number",
      "condition": "string"
    }
  ]
}
```

Simulate a weather API response for Paris today.”

## Output

```json
{
  "location": "Paris",
  "temperature_celsius": 18,
  "condition": "Cloudy",
  "humidity_percent": 72,
  "wind_kph": 15,
  "forecast": [
    { "time": "09:00", "temperature_celsius": 16, "condition": "Cloudy" },
    { "time": "12:00", "temperature_celsius": 18, "condition": "Partly Sunny" },
    { "time": "15:00", "temperature_celsius": 19, "condition": "Sunny" },
    { "time": "18:00", "temperature_celsius": 17, "condition": "Cloudy" }
  ]
}
```
