# Chaining Prompts for Agent Flow

## Step 1: Ask for User Query

### Prompt:

“Please enter your weather-related question.”

User Input Example:

“What’s the temperature in Paris?”

### Output:

Captured user query: "What’s the temperature in Paris?"

## Step 2: Simulate Weather API Call

### Prompt:

“Given the weather JSON format:

```json
{
  "location": "string",
  "temperature_celsius": "number",
  "condition": "string",
  "humidity_percent": "number",
  "wind_kph": "number",
  "forecast": [
    {
      "time": "string",
      "temperature_celsius": "number",
      "condition": "string"
    }
  ]
}
```

Simulate a weather API response for Paris today.”

### Simulated API Output:

```json
{
  "location": "Paris",
  "temperature_celsius": 18,
  "condition": "Cloudy",
  "humidity_percent": 72,
  "wind_kph": 15,
  "forecast": [
    { "time": "09:00", "temperature_celsius": 16, "condition": "Cloudy" },
    { "time": "12:00", "temperature_celsius": 18, "condition": "Partly Sunny" },
    { "time": "15:00", "temperature_celsius": 19, "condition": "Sunny" },
    { "time": "18:00", "temperature_celsius": 17, "condition": "Cloudy" }
  ]
}
```

## Step 3: Format Final Answer

### Prompt:

“Format the weather API output into a concise user-friendly answer.”

### Formatted Output:

“Today in Paris, it’s 18°C and cloudy. Humidity is 72% with winds at 15 kph. Forecast: 09:00 – 16°C Cloudy, 12:00 – 18°C Partly Sunny, 15:00 – 19°C Sunny, 18:00 – 17°C Cloudy.”

This sequence captures:

User query → 2. API simulation → 3. User-friendly answer
